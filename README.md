# Bilibili VIP Quality Filter

> **想象一下：**
> 深夜，你正惬意地躺在椅子上，准备享受一段精彩的 B 站视频。你习惯性地打开清晰度菜单，想切换到更高画质。鼠标滑过列表，你自信地点击了那个看起来最清晰的选项——
>
> **突然，视频暂停了，屏幕中央弹出一个巨大的窗口：“该清晰度仅限大会员使用”。**
>
> 那一刻，沉浸感瞬间破碎，只剩下被打断的恼火。
>
> 如果你也受够了这种误触带来的挫败感，这个脚本就是为你准备的。它会默默地帮你“打扫”播放器，把那些你无法使用的 VIP 选项统统藏起来，只留下属于你的纯粹观看体验。

这是一个简单的油猴脚本，用于在哔哩哔哩视频播放器中自动过滤大会员专属清晰度选项。

## 📥 安装

**[👉 点击此处直接安装脚本](https://github.com/dongyu23/Bilibili-VIP-Quality-Filter/raw/refs/heads/main/bilibili-vip-quality-filter.user.js)**

## 功能

- 自动识别播放器中的清晰度菜单项（`bpx-player-ctrl-quality-menu-item`）。
- 检查其中是否包含“大会员”关键字。
- 如果包含，则自动移除该选项，避免误触或干扰。
